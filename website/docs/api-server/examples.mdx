### Create Pull Request Example

This example demostrates rendering a template to a Github repository.

It shows off:
- Logging into the WGE API
- Exchanging a local `GITHUB_TOKEN` for a JWT token we can then use to do git operations via the WGE API
- Rendering a template and writing the result to a Pull Request
